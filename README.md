This is a simple recipe website made using HTML and CSS. It has multiple pages and each page shows a different recipe. The website is made for my web design assignment and it is responsive, so it can open properly on desktop, tablet, and mobile screens.
FILES INCLUDE:
- Home
- Poha
- Rajma Chawal
- Chole Bhature
- Butter Chicken with Garlic Naan
- Moong Daal Halwa
FEATURES ADDED AS PER INSTRUCTIONS:
- Multi-page website
- Navigation bar on all pages
- Separate recipe page for each dish
- CSS styling for layout, colors, spacing, and fonts
- Responsive design using media queries
- Images and recipe details on each page
- Simple and clean user interface
The purpose of this project is to create a basic multi-page website using only HTML and CSS. It shows different food recipes in a simple and organized way.
This website is deployed using GitHub Pages.
